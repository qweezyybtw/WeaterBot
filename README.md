Бот для телеграма для получения погоды.

Чтобы его использовать нужно найти бота по тэгу @ltzFlikChan_TWbot в телеграме и написать /start.
Для просмотра погоду нужно либо написать координаты города или места, где вы хотите посмотреть погоду, либо название города.
Кнопка "О проэкте" или же команда /about пишет все о создателе проэкта (на самом деле нет).
