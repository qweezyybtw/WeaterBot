Бот для телеграма для получения погоды.

Чтобы его использовать нужно в телеграме и написать /start.
Для просмотра погоду нужно либо написать координаты города или места, где вы хотите посмотреть погоду, либо название города.
Кнопка "О проэкте" или же команда /about пишет все о создателе проэкта.

Бот использует [openweatherapi.com](https://api.openweathermap.org) для получения погоды. Он делает запросы и отправляет влажность, температуру, как ощущается и саму погоду.
Чтобы его использовать замените TOKEN и WEATHER_TOKEN на свой токен. TOKEN - это токен бота телеграма, WEATHER_TOKEN - это токен [openweatherapi.com](https://api.openweathermap.org).
